# Univ_Chatbot
## ERUDA – 대학 학사정보 맞춤형 챗봇

## 📌 프로젝트 개요
**ERUDA(이루다)**는 대학 학사 공지를 기반으로 학사 정보를 자동으로 제공하는 챗봇입니다.  
사용자는 자연어로 질문을 입력하면 학사 공지를 바탕으로 적절한 답변을 받을 수 있습니다.

## 🏫 대상 사용자
- 대학 구성원 (학생, 교수, 행정 직원 등)

## 🎯 주요 기능
- 대학 학사 공지 자동 수집 및 분석
- 자연어 질문에 대한 적절한 학사 정보 제공 (RAG 활용)
- 최신 공지 반영을 위한 정기 업데이트
